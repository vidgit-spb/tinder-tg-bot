### 3. 📸 **Ограничение размера фото (10 МБ)**

**Реализовано:**
- Максимальный размер фото: **10 МБ**
- Проверка при загрузке фото (регистрация и редактирование)
- Пользователь видит текущий размер и ошибку если превышен лимит

**Код проверки:**
```python
photo = update.message.photo[-1]
file = await context.bot.get_file(photo.file_id)
file_size_mb = file.file_size / (1024 * 1024)

if file_size_mb > 10:
    await update.message.reply_text(
        f"❌ Фото слишком большое! Максимум 10 МБ.\n"
        f"Текущий размер: {file_size_mb:.1f} МБ"
    )
    return
```

**Примечание о "максимум 3 фото":**
- Telegram Bot API хранит фото на серверах Telegram
- Бот сохраняет только `file_id` (текстовый идентификатор)
- Фото не занимают место на вашем сервере!
- Можно добавить поддержку нескольких фото, но сейчас реализовано 1 фото

### 4. 📊 **Система статистики**

**Новые таблицы в БД:**

#### `profile_views` - просмотры профилей
```sql
- viewer_id (кто смотрел)
- viewed_id (чей профиль)
- created_at (когда)
```

#### `user_statistics` - статистика пользователя
```sql
- user_id
- total_views (сколько раз смотрели профиль)
- total_likes_sent (отправлено лайков)
- total_likes_received (получено лайков)
- total_matches (количество матчей)
- total_messages_sent (отправлено сообщений)
- total_messages_received (получено сообщений)
- last_active (последняя активность)
```

**Автоматическое отслеживание:**
- ✅ Просмотр профиля → `track_profile_view()`
- ✅ Лайк → `update_like_stats()`
- ✅ Матч → `update_match_stats()`
- ✅ Сообщение → `update_message_stats()`

## 🎯 Как использовать

### Для пользователей:

#### Установить фильтр возраста:
1. Главное меню → ⚙️ Settings
2. Нажать на "🎂 Возраст: X-Y лет"
3. Выбрать диапазон
4. Начать свайпинг - будут показаны только профили в этом диапазоне

#### Загрузить фото:
- При регистрации или редактировании профиля
- Если фото > 10 МБ - появится ошибка с текущим размером
- Нужно сжать фото или выбрать другое

### Для админов:

#### Посмотреть статистику:
1. Отправить `/admin` в чат с ботом
2. Увидеть общую статистику и топ-10
3. Нажать "📊 Детальная статистика" для подробностей
4. Нажать "🔄 Обновить" для обновления данных

#### Добавить себя в админы:
В `config.py`:
```python
ADMIN_USER_IDS = [431713422]  # Ваш Telegram user ID
```

Узнать свой ID можно через бота @userinfobot
