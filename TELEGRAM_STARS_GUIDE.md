# ⭐ Полное руководство по Telegram Stars

## 🎯 Что такое Telegram Stars

**Telegram Stars** — встроенная валюта Telegram для оплаты внутри ботов.

- 1 Star ≈ $0.013 (1.3 цента)
- Пользователи покупают Stars в Telegram
- Платят ими в вашем боте
- Вы получаете деньги на баланс бота

---

## 💰 Куда идут деньги

### Схема платежа:

```
1. Пользователь покупает Stars в Telegram
   ↓
2. Платит Stars в вашем боте (например, 1 Star)
   ↓
3. Деньги зачисляются на баланс ВАШЕГО БОТА
   ↓
4. Вы выводите деньги себе на карту/кошелек
```

**Важно:** Деньги идут **ВАМ**, не Telegram (Telegram берёт комиссию ~30%)

---

## 🔧 Настройка платежей (пошагово)

### Шаг 1: Активация Telegram Stars в @BotFather

1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте: `/mybots`
4. Выберите вашего бота из списка
5. Нажмите **"Bot Settings"**
6. Нажмите **"Payments"**
7. Выберите **"Telegram Stars"**
8. Прочитайте и согласитесь с условиями
9. Нажмите **"Enable"**

✅ **Готово!** Ваш бот теперь может принимать Stars.

---

### Шаг 2: Проверка работы платежей

**Важно:** Платежи работают **ТОЛЬКО на продакшене** (после деплоя на Railway/Render).

**Локально платежи НЕ работают!**

После деплоя:
1. Откройте бота в Telegram
2. Попробуйте функцию "Who Liked Me"
3. Нажмите "Unlock for 1 Star"
4. Telegram покажет окно оплаты
5. Оплатите (для теста)

## 💳 Вывод денег

### Когда можно выводить:

- Минимальная сумма: **1000 Stars** (~$13)
- Вывод через @BotFather

### Способы вывода:

#### 1. TON Wallet (криптокошелек) ⚡ БЫСТРО

**Преимущества:**
- Мгновенный вывод
- Низкая комиссия
- Анонимно

**Как настроить:**
1. Установите **Tonkeeper** или **Tonhub** (приложения для TON)
2. Создайте кошелек
3. Скопируйте адрес кошелька
4. В @BotFather → Payments → Withdraw → TON Wallet
5. Вставьте адрес
6. Подтвердите

**Конвертация:**
- TON можно обменять на USDT/BTC на биржах
- Или вывести на карту через P2P

---

#### 2. Fragment (платформа Telegram) 💎

**Преимущества:**
- Официальная платформа Telegram
- Можно продать Stars за TON
- Безопасно

**Как:**
1. @BotFather → Payments → Withdraw
2. Выберите "Fragment"
3. Stars переведутся на Fragment.com
4. Продайте их за TON
5. Выведите TON на кошелек

---

#### 3. Банковская карта 💳 (через партнёров)

**Через сервисы:**
- Некоторые страны поддерживают прямой вывод
- Проверьте доступность в @BotFather

```python

2. Сохранение в базу данных
3. Защита от повторной оплаты

```

### Таблица payments в БД:

```sql
CREATE TABLE payments (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,           -- Кто заплатил
    stars_amount INTEGER,      -- Сколько Stars
    feature TEXT,              -- За что (see_likes)
    payment_id TEXT,           -- ID транзакции Telegram
    created_at TIMESTAMP       -- Когда
)
```

## 📊 Аналитика платежей

### Проверка статистики:

```bash
# Подключитесь к базе данных:
sqlite3 dating_bot.db

# Общий доход:
SELECT SUM(stars_amount) FROM payments;

# Количество покупок:
SELECT COUNT(*) FROM payments;

# Топ покупатели:
SELECT user_id, COUNT(*) as purchases 
FROM payments 
GROUP BY user_id 
ORDER BY purchases DESC 
LIMIT 10;

# Доход по дням:
SELECT DATE(created_at) as date, SUM(stars_amount) as stars
FROM payments
GROUP BY DATE(created_at)
ORDER BY date DESC;
```

## 🆘 Частые вопросы

**Q: Когда я получу деньги?**
A: Сразу после оплаты пользователем. Баланс в @BotFather обновляется мгновенно.

**Q: Какая комиссия Telegram?**
A: ~30% от суммы платежа.

**Q: Можно ли вернуть деньги пользователю?**
A: Нет, Telegram Stars не возвращаются.

**Q: Минимальная сумма вывода?**
A: 1000 Stars (~$13).

**Q: Как часто можно выводить?**
A: Без ограничений, как только достигнут минимум.

**Q: Нужна ли регистрация ИП/ООО?**
A: Зависит от вашей страны и объёмов. Проконсультируйтесь с юристом.
